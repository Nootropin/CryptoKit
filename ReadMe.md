# What is the project
The project's purpose is to make easy to use GUI tool for encrypting, cyphering, hashing and other cryptographic functions in simple GUI interface
# Building 
```bash
git clone https://github.com/Nootropin/CryptoKit
cd CryptoKit
mkdir build
cd build
cmake ..
make -j4
```
